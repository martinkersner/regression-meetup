# Linear Regression

Martin Kersner, <m.kersner@gmail.com>

## Clone repository
```bash
git clone https://github.com/martinkersner/regression-meetup
```

## Run Jupyter notebook
```bash
virtualenv venv -p `which python3`
source venv/bin/activate
pip3 install -r requirements.txt
jupyter notebook
```
